Geometrical constructor from a XML format reading.

Reading data from XML file under the following format: ex : <dataRectangle axe1="x" axe2="y" laxe1="0.3" laxe2="0.2"> <posInferiorVertex x="0.4" y="0.5" z="0."> </dataRectangle>
